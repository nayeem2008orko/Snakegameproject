# 🐍 Snake Game (Pygame)

A fun Snake game built with Pygame, featuring:

- Maze walls
- Score and highscore tracking
- Game Over screen with restart and exit options
- Custom graphics

## 🕹 Controls

- W / A / S / D — move the snake
- F — restart after game over
- X — exit the game

## ▶ How to Run

```bash
pip install pygame
python main.py
